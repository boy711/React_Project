# StudentSpace Store

StudentSpace Store is a Create React App style ecommerce website. It uses React, React Router DOM, Context API, plain CSS, Fetch API, and the DummyJSON products API.

## Features

- Home, Shop, Product Details, About, Contact, Cart, Login, and Signup pages
- Products fetched from `https://dummyjson.com/products`
- Single product details fetched from `https://dummyjson.com/products/:id`
- Search, category filter, and price sorting
- Cart state with localStorage persistence
- Fake authentication with localStorage persistence
- Responsive layout for mobile, tablet, and desktop

## Install

```bash
npm install
```

## Run

```bash
npm start
```

The app will open at `http://localhost:3000`.

## Build

```bash
npm run build
```
